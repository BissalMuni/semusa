# 2022년 제14번

이자소득세 부과의 효과에 관한 내용으로 옳은 것의 개수는? (단, 현재소비와 미 래소비는 모두 정상재이다.)

① 1개
② 2개
③ 3개
④ 4개
⑤ 5개

## 기본 해설

이자소득세→세후이자율↓→현재소비 상대가격↓. 대체효과: 현재소비↑(저축↓). 소득효과: 실질소득↓→현재소비↓(저축↑). 순효과 불확실.

## 상세 해설

【시점간 소비선택과 이자소득세】
세후이자율: r(1-t)
대체효과: 이자율↓→현재소비↑, 미래소비↓→저축↓
소득효과: 실질소득↓→현재소비↓, 미래소비↓→저축↑

정상재 가정에서:
- 대체효과와 소득효과가 저축에 반대 방향
- 순효과 불확실
- 저축자(lender)의 경우에 해당
