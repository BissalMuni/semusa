# 2023년 제21번

두 소비자 1과 2가 두 재화 x와 y를 소비하는 순수교환경제를 고려하자. 소비자 1의 부존은 (18, 12), 소비자 2의 부존은 (22, 8)이며, 소비자 i의 효용함수는 u_i(x_i, y_i) = x_i·y_i라 하자(i = 1, 2). 다음의 배분 ((x_1, y_1), (x_2, y_2)) 중 파레토 개선이 가능하지 않은 것을 모두 고르면?

① ㄱ, ㄴ
② ㄱ, ㄹ
③ ㄴ, ㄷ
④ ㄱ, ㄴ, ㄹ
⑤ ㄴ, ㄷ, ㄹ

## 보기

- ㄱ. ((6, 3), (34, 17))
- ㄴ. ((16, 8), (22, 11))
- ㄷ. ((24, 10), (16, 10))
- ㄹ. ((38, 19), (2, 1))

## 기본 해설

규모의 경제→한 기업이 전체 시장 공급이 효율적. MC가격=효율적 자원배분, but 적자. AC가격=영이윤, but P>MC.

## 상세 해설

【자연독점 규제】
MC가격: P=MC<AC→적자→정부보조 필요
AC가격: P=AC→영이윤→비용절감유인 없음
이부가격: 기본료+사용료(=MC)→효율+적자보전
램지가격: 역탄력성 비례→DWL최소

규제포획이론: 규제기관이 피규제자에 포획
